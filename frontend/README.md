# HomeSense Frontend

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```

## Features
- User authentication via AWS Amplify/Cognito
- Device dashboard (real-time status)
- Control panels for devices
- Automation rules UI
- Notification center
